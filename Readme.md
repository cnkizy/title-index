---

![](title-index/example_title_index.png)  
![](title-index/Display.png)

使用场景:导航、下载

该组件有两种状态方案：
1.鼠标经过改变状态
2.鼠标点击改变状态

目前有两个版本 一个原生js版，一个Vue版.
